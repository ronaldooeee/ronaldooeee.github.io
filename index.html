ScriptoFino is a simple scripting language that draws inspiration from Python. It aims at removing the cognitive dissonance between thinking in code and writing out its implementation. Additionally, SciptoFino is designed to mimic the natural syntax and structure of the Spanish language.

Geometric Modeling of a Scene

A replication of a real world scene using hierarchical modeling of the objects. Each object was created by building prisms from planes using 3D translations and rotations. These prisms are then used to form parts of objects. S hading is then added to these objects by defining a light source, computing the surface normals, and generating the observed colors using Gouraud shading.
